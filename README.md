## Genèse et Vision

L'essence de **CodeKnow** réside dans la fusion de l'apprentissage technique et de l'esthétique liminale des jeux de rôle. Dans cet univers, la maîtrise du langage machine ne s’acquiert point par la théorie aride, mais par la complétion de quêtes interactives, sous l'égide de mentors dotés d'une intelligence artificielle avancée.

### Les Piliers de l'Académie

*   **Quêtes de Haute Volée** : Des épreuves méticuleusement conçues pour éprouver la logique et la syntaxe, de l'initiation à la maîtrise architecturale.
*   **Les Gardiens de l'Index** : Sept entités distinctes — de Rin la perfectionniste du pixel à Nova la voyageuse dimensionnelle — guident l'étudiant avec une rigueur et une personnalité qui leur sont propres.
*   **Exégèse Algorithmique** : Grâce à l'intégration du modèle **Gemini 3 Flash**, chaque soumission fait l'objet d'une analyse sémantique profonde, prodiguant un feedback à la fois technique et philosophique.

## Architecture Technique

La structure de CodeKnow repose sur une synergie de technologies de pointe, garantissant une expérience fluide et pérenne :

*   **Frontispice** : Une interface réactive bâtie sous **React 18** et **Vite**, sublimée par **TypeScript**.
*   **Intelligence Artificielle** : Le déploiement du SDK **Google Generative AI** permet une interaction dynamique et contextuelle.
*   **Persistance et Authentification** : L'écosystème **Firebase** assure la sauvegarde immuable de la progression et des grades de chaque utilisateur.
*   **Esthétique Immersive** : Un design inspiré du minimalisme cyberpunk, employant des filtres CRT et des animations de transitions fluides.

## Protocole d'Installation

### Prérequis Systémiques
*   Environnement d'exécution **Node.js** (v18+)
*   Une clé d'accès cryptographique à l'API **Gemini**

### Déploiement Local

1.  **Extraction du Répertoire**
    ```bash
    git clone https://github.com/Jonathan-MOUELE/CodeKnow.git
    cd CodeKnow
    ```

2.  **Initialisation des Dépendances**
    ```bash
    npm install
    ```

3.  **Configuration de l'Environnement**
    Établissez un fichier `.env.local` à la racine du projet, en y renseignant vos identifiants Firebase et votre clé API Gemini :
    ```env
    # Firebase Configuration
    VITE_FIREBASE_API_KEY=votre_identifiant
    # ... autres paramètres Firebase
    
    # Intelligence Artificielle
    GEMINI_API_KEY=votre_cle_gemini
    ```

4.  **Lancement du Serveur**
    ```bash
    npm run dev
    ```
